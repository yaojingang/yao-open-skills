# Intent Dialogue

Use these questions before deep research. Stop once the tutorial can be described clearly in one sentence.

## Five Sharp Questions

1. What exact topic should the tutorial teach?
2. Who is the learner and what is their current level?
3. What concrete outcome should the learner reach after finishing the tutorial?
4. What time, format, or source constraints matter?
5. Should the output stop at markdown, or also produce `docx`, `pdf`, and an optional webpage?

## Nice-To-Have Questions

- What examples should feel familiar to the learner?
- What should stay out of scope for the first version?

## First-Pass Assumptions For This Skill

- recurring job: build a personalized tutorial from authoritative sources
- real inputs: topic, learner profile, target outcome, time budget, required sources, output formats
- required outputs: markdown tutorial, source appendix, optional `docx`, optional `pdf`, optional webpage follow-up
- exclusions: generic writing, unsourced summaries, one-off fact lookup, standalone webpage work
- recommended archetype: `production`
- recommended first eval target: distinguish a real tutorial-generation request from generic research or plain document export
